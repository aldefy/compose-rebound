# AI Agent Skill

Make your AI coding tool understand Compose recomposition performance.

The **[rebound-skill](https://github.com/aldefy/rebound-skill)** repo contains markdown-based instructions that any AI assistant can consume. It covers budget classes, violation diagnosis, skip rate analysis, stability optimization, CLI usage, and IDE plugin workflow.

## Supported Tools

Claude Code, Gemini CLI, Gemini (Android Studio), Cursor, GitHub Copilot, Codex CLI, Windsurf, Amazon Q Developer, and any tool that reads markdown instructions.

## Setup

Visit the [rebound-skill repo](https://github.com/aldefy/rebound-skill) for tool-specific setup instructions.

The general pattern: copy `rebound-expert-skill/` into your project or tool config directory, then point your tool's instruction file at `rebound-expert-skill/SKILL.md`.
